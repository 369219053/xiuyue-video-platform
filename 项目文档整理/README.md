# 秀悦AI视频自动化平台

## 📖 项目简介

甲方每天提供一份 Excel 表格（包含多个组的视频拍摄需求），本项目将整个流程自动化：
**解析 Excel → 在飞书建副表 → 调用 Coze 工作流批量生成视频 → 将 debug_url 写入飞书台账**

单人操作，每天处理几百条 AI 投流视频（每条约 10 秒），有约 30 小时的处理时间窗口。

## 🚀 当前操作流程（一键自动化）

```
1. 运行平台
   python3 src/app_server.py
   → 弹出桌面窗口（PyWebView）

2. 在界面上传甲方 Excel
   → 系统自动解析，按 日期 → 组名 → 产品 分层展示

3. 填写配置（首次填写后自动保存，下次自动还原）：
   - Coze Token
   - 工作流 ID
   - 飞书 App ID + App Secret（用于写台账）
   - 飞书 Base Token（副表所在 Bitable 的 token）

4. 选择要执行的产品，点击「⚡ 确认，开始执行」
   系统自动完成三步：
   Step 1/3 🗂️ 飞书 CDP 建副表（把数据写入飞书多维表格）
   Step 2/3 🤖 并行触发 Coze 工作流（最多5个同时，收集 debug_url）
   Step 3/3 📋 写入飞书台账（将 任务名称 + debug_url 记录到台账表）
```

## ⚠️ 飞书权限配置（重要！首次使用必做）

飞书自建应用需要手动授权才能写入多维表格：

1. 打开 [飞书开放平台](https://open.feishu.cn/app) → 找到自建应用
2. 「权限管理」→ 搜索 `base:record:create` → 开通
3. 打开台账多维表格 → 右上角「···」→「添加应用」→ 搜索自建应用 → **权限设为"可编辑"**
4. 副表 Bitable 同样需要执行步骤3（如果 Step1 建副表失败）
